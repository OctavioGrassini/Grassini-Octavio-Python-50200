Lema Libros

Ésta página es un prototipo de web, realizado con Django, como proyecto final al curso de Python de Coderhouse, comisión 50200.
El proyecto se centra en la parte de backend de la página, por lo que los aspectos de HTML y CSS se muestran descuidados, pero lo suficientemente funcionales y estéticos para que pueda evaluarse el proyecto con comodidad.
El prototipo se basa en una página para gestión de una libreria en la cual además de información de libros, es necesario registrar las editoriales, los clientes y hacer reservas.

Se detalla más información del proyecto y usuarios disponibles en el archivo README.txt adjunto.

Además, se encuentra disponible un video de introducción al proyecto y un link a una página "acerca de mí"
